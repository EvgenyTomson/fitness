- Redux-toolkit
- axios
- Vite
- react-intl

### Установка и запуск

Установить Node.js  
Node Version Manager

```bash
nvm install
```

Fast Node Manager

```bash
fnm install
fnm use
```

Установить зависимости:

```bash
npm install
```

Сбилдить проложение:

```bash
npm run dev
```
